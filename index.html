How I work

Find the real problem

At Shopee, the bottleneck was not query speed. Metric definitions, analytical logic, recurring workflows, and team knowledge had to be standardized before they could be automated reliably. Diagnosis comes before solution.

Automate execution, never understanding

People may hand execution to AI, but they must still see why. Aesthetics, evaluation, and decisions stay human; uncertain cases go back to a person.

Constrain what must not fail

SQL templates, output schemas, lint rules, explicit failure modes — deterministic code guarantees what the model cannot.
